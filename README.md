## Autonomous Snowplow - ROS2 - Humble Edition

## About Snow Plow

Autonomous Snow Plow:  Autonomous Snowplow is tasked with clearing a pre-determined area of snow without any human input; we can only use algorithms, computer vision, and additional sensors. We recently rebuilt the drive train.
